Security Statement

When MI Online is accessed, it invokes SSL (Secure Socket Layers) encryption. SSL uses public-key encryption to exchange a session key between the client (your PC) and Radian's server. The session key encrypts the data submitted to and requested from MI Online. Each transaction uses a different session key so that even if a session key were decrypted, it would not provide access to the server, but only to a small piece of the transmitted information.

MI Online session keys use 128-bit key encryption, meaning there are 2128 possible combinations. Using today's best technology, the odds of decrypting a session key are near impossible.
